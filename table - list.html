Tech Glossary

HTML
HTML stands for HyperText Markup Language, the standard language used to create and structure content on the web. It uses special codes called tags to tell web browsers how to display text, images, and other media on a webpage. HTML provides the fundamental structure for web pages, with Cascading Style Sheets (CSS) used for styling and JavaScript for interactive features to build modern websites.
CSS
CSS, or Cascading Style Sheets, is a style sheet language used in web development to describe the presentation of a document written in HTML or XML. It allows developers to control the visual appearance of web pages, including elements such as colors, fonts, layout, and spacing.
Java Script
JS is the common abbreviation for JavaScript. JavaScript is a high-level, interpreted programming language that is one of the three core technologies of the World Wide Web, alongside HTML and CSS.
REACT
React, also known as React.js or ReactJS, is an open-source JavaScript library primarily used for building user interfaces (UIs) for web applications. It was developed by Facebook and is widely adopted for its efficiency and declarative approach to UI development.
